## Setup

`Ruby -v 3.2.2`
`Rails -v 7.0.7`

Install your bundle by run 'bundle install'

Create database with `rails db:create`

For node_modules run `yarn install`

For assets compile use `rails assets:precompile`

Now run this project by `rails s`
